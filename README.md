# 🌐 ADA Lab Assignment 3 - Graph Algorithms

## 📌 Overview
This repository contains implementations of important graph algorithms used in computer science and real-world applications like navigation systems, social networks, and scheduling.

---

## 🧠 Topics Covered

### 🔹 Graph Representation
- Adjacency List
- Adjacency Matrix

### 🔹 Graph Traversal
- Breadth First Search (BFS)
- Depth First Search (DFS)

### 🔹 Topological Sorting
- Task scheduling using DAG

### 🔹 Shortest Path Algorithms
- Dijkstra’s Algorithm
- Bellman-Ford Algorithm

### 🔹 Minimum Spanning Tree (MST)
- Prim’s Algorithm
- Kruskal’s Algorithm
